1. Java と C++ の違い
 C++とjavaはオブジェクト指向だが、C++は派生元であるⅭ言語であるⅭ言語との互換性があり、Java換性ある言語なし。実行速度はⅭ++のほうが速い。

2. Java と JavaScript の違い
Javaは多くの種類のソフトウェア開発ができ応用きく。JVMでOSに依存せず、どんなコンピューターでも動く。JavaScriptはWEBブラウザ、テキストエディタだけで動く。
文法が違う。Javaは動的型付　JavaScriptは静的型付
　
3. Java と Python の違い
文法が違う。Javaは動的型付　コンパイル必要。(データを最初に固定)
Pythonは静的型付　コンパイル不要。(データを意識しなくていい）
ライブラリ(ある程度まとまった機能を提供するプログラミングの部品。ある処理をしたいときに必要なところだく使う）異なる。Javaは汎用性、Pythonは数値処理に特化。
